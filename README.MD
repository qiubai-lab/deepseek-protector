## 2.项目初始化与构建

1. **克隆项目：**
```bash
git clone https://github.com/qiubai-lab/deepseek-protecto.git
cd deepseek-protecto
```

2. **安装依赖：**
```bash
go mod tidy
```

3. **使用以下命令构建项目：**
```bash
go build -o deepseek-protector ./cmd/main.go
```

构建完成后，您可以在项目根目录下找到生成的可执行文件 `deepseek-protector`。
